This is a basic system that uses Phaxio's API to send faxes. To set it up for yourself or your own friend group, you'll need a developer account from Phaxio, which is out of scope of this document.

Once you have a funded Phaxio account, you'll need two codes from Phaxio's API Keys and one from Webhooks.

Account Credentials

You'll need your "Account SID" and "Auth Token", both of which are also in the console. These are secrets, so be sure not to share them with anyone! The values in .env are only visible to the owner of the project, so you can safely paste them there: 

# find these at https://console.phaxio.com/api_credentials
PHAXIOKEY=XXXXXXXXXXXXXXXXXXXXXXXXXXXX
PHAXIOSECRET=XXXXXXXXXXXXXXXXXXXXXXXXXXXX

# find this at https://console.phaxio.com/user/callbacks/edit
PHAXIOTOKEN=XXXXXXXXXXXXXXXXXXXXXXXXXXXX

A Secret!

This app will send a real fax to a phone number and charge you money for doing so, and we don't want just anyone to be able to do that! Pick out a secret phrase or password you can use to make sure only you and your friends can send a fax from the app, and add it to .env 

# let's not let just anyone fax you
SECRET=XXXXXXXX
