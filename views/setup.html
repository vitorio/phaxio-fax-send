Twilio SMS Demo Setup

You're on your way!

This is a basic example of how to use Twilio's SMS API to send texts. To continue, you'll need a developer account from Twilio.

sign up for a twilio account

While signing up, you can streamline things by selecting that you write code, you want to "Explore Twilio", and you want to send and recieve SMS messages.

Your Phone Number

Along the way, they'll ask for a phone number. You'll also enter that phone number into the .env file of this project as the variable YOUR_PHONE_NUMBER. For example, if your phone number is (US) 555-867-5309, you'll put the following in .env: 

# This is the phone number where test messages will be sent.
# Twilio will have asked you to enter and verify this!
YOUR_PHONE_NUMBER="+15558675309"

For your remix to run, you'll need a few pieces of information from Twilio.

Twilio Phone Number

Twilio will provide you a phone number your code can use to send message from. You select via "Get A Trial Number" in your Twilio Developer Console This also goes into the .env file of this project: 

# find these at https://www.twilio.com/console
# TWILIO_PHONE_NUMBER="+15551234567"
# TWILIO_ACCOUNT_SID=XXXXXXXXXXXXXXXXXXXXXXXXXXXX
# TWILIO_AUTH_TOKEN=XXXXXXXXXXXXXXXXXXXXXXXXXXXX

Account Credentials

You'll also need your "Account SID" and "Auth Token", both of which are also in the console. These are secrets, so be sure not to share them with anyone! The values in .env are only visible to the owner of the project, so you can safely paste them there: 

# find these at https://www.twilio.com/console
# TWILIO_PHONE_NUMBER="+15551234567"
# TWILIO_ACCOUNT_SID=XXXXXXXXXXXXXXXXXXXXXXXXXXXX
# TWILIO_AUTH_TOKEN=XXXXXXXXXXXXXXXXXXXXXXXXXXXX
