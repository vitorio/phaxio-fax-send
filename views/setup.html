Twilio SMS Demo Setup

You're on your way!

We need a few more pieces of information to finish setting up the app. This page will track your progress and show which steps you've completed with a checkmark (✅).

This is a basic example of how to use Twilio's SMS API to send texts. To continue, you'll need a developer account from Twilio. Already have one? just continue below.

sign up for a twilio account

While signing up, you can streamline things by selecting that you write code, you want to "Explore Twilio", and you want to send and recieve SMS messages.

Your Phone Number

Along the way, they'll ask for a phone number. You'll also enter that phone number into the .env file of this project as the variable YOUR_PHONE_NUMBER. For example, if your phone number is (US) 555-867-5309, you'll put the following in .env: 

# This is the phone number where test messages will be sent.
# Twilio will have asked you to enter and verify this!
YOUR_PHONE_NUMBER="+15558675309"

For your remix to run, you'll also need a few pieces of information from Twilio. You can find them in the Twilio Developer Console.

Twilio Phone Number

Twilio will provide you a phone number your code can use to send message from. You select via "Get A Trial Number" in the developer console. This also goes into the .env file of this project: 

# find these at https://www.twilio.com/console
# TWILIO_PHONE_NUMBER="+15551234567"
# TWILIO_ACCOUNT_SID=XXXXXXXXXXXXXXXXXXXXXXXXXXXX
# TWILIO_AUTH_TOKEN=XXXXXXXXXXXXXXXXXXXXXXXXXXXX

Account Credentials

You'll also need your "Account SID" and "Auth Token", both of which are also in the console. These are secrets, so be sure not to share them with anyone! The values in .env are only visible to the owner of the project, so you can safely paste them there: 

# find these at https://www.twilio.com/console
# TWILIO_PHONE_NUMBER="+15551234567"
# TWILIO_ACCOUNT_SID=XXXXXXXXXXXXXXXXXXXXXXXXXXXX
# TWILIO_AUTH_TOKEN=XXXXXXXXXXXXXXXXXXXXXXXXXXXX

A Secret!

This app will send an SMS to a phone number, and we don't want just anyone to be able to do that! Pick out a secret phrase or password you can use to make sure only you can send a message from the app, and add it to .env 

# let's not let just anyone text you
SECRET=XXXXXXXX
