Twilio SMS Demo Setup

You're on your way!

This is a basic example of how to use Twilio's SMS API to send texts. To continue, you'll need a developer account from Twilio.

sign up for a twilio account

Along the way, they'll ask for a phone number. You'll also enter that phone number into the .env file of this project as the variable YOUR_PHONE_NUMBER. For example, if your phone number is (US) 555-867-5309, you'll put the following in .env: 

# This is the phone number where test messages will be sent.
# Twilio will have asked you to enter and verify this!
YOUR_PHONE_NUMBER="+15558675309"

For your remix to run, you'll need a few pieces of information from Twilio.
